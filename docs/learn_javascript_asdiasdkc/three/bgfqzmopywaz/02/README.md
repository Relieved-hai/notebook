## 像素:px

像素 `px` 是最基本，绝对和最终的度量单位。

像素数是在 [屏幕分辨率](https://en.wikipedia.org/wiki/Display_resolution) 设置中设置的，一个 `px` 只是屏幕上的此类像素之一。浏览器最终会将所有值转换为像素。

像素可以是小数，例如，大小可以设置为 `16.5px`。这是完全正常的，浏览器本身使用小数像素进行内部计算。例如，有一个宽度为 `100px` 的元素，需要将其分为三部分，`33.333...px` 出现。在最终渲染中，小数像素当然会被四舍五入并制成整体。

对于屏幕上像素很多但屏幕本身很小的移动设备，浏览器将自动应用缩放以确保可读性。

- 优点

  - 像素的主要优点是清晰度和清晰度

- 缺点

  - 其他度量单位-从某种意义上讲“更强大”，它们是相对的，可让您确定不同大小之间的比率

<br/>

:::warning
**很久以前:`mm`, `cm`, `pt`, `pc`**

但它们早已被抛弃。

如果你感兴趣的话，它们的值是:

`1mm`（mm）= `3.8px`
`1cm`（cm）= `38px`
`1pt`（点）= `4/3 px`
`1pc`（派卡）= `16px`

由于浏览器将这些值重新计算为像素，因此没有必要使用它们。
:::

<br/>

:::tip
**为什么cm的厘米正好是38像素?**

在现实生活中，厘米是长度的标准，十分之一米。 [像素](https://en.wikipedia.org/wiki/Pixel) 可能会有所不同，具体取决于屏幕。

但是在以上公式中，像素应理解为“真空中的球形像素”，即“标准化屏幕”上的一点，其特点在 [规格](https://www.w3.org/TR/CSS2/syndata.html#length-units) 说明。
:::

## 关于字体:em

`1em` 是目前字体的大小。

你可以从当前字体中选择任何比例:`2em`, `0.5em` 等等。

**`em` 的大小是相对的，它们是根据当前上下文环境决定的。**

例如，让我们把 `px` 和 `em` 比较一下。

```html
<div style="font-size:24px">
  鸵鸟
  <div style="font-size:24px">它们生活在非洲</div>
</div>
```
