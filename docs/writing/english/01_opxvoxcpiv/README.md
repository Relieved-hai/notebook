从名字上看，没有母就没有子，所以在英语中，母音比较重要。所以我们要找到一个单词中的母音

- **qu**: qu 一定是在一起的

- **y**: 如果放在辅音的后面，它就是元音，如果在元音的前面就是辅音

- **i y**: 它们两个经常在一起

- 如果只有一个音节的时候，这音节读四声，有两个音节，那么开头的音节读重音，也就是拼音的一声


<br/>

看见单词，从后往前，先找音节(找元音)，然后和前面紧挨着的一个辅音和在一起，组成一个音节，这个音节用汉语四声来读
